## ⚙️ Requirements

Antes de ejecutar el proyecto, asegúrate de tener instalado **Python 3.8+** y los siguientes paquetes:

| Paquete | Descripción | Versión recomendada |
|----------|--------------|--------------------|
| **Flask** | Framework web ligero para Python | `3.0.0` |
| **requests** | Cliente HTTP para realizar peticiones a GraphDB y Wikidata | `2.31.0` |

Instálalos ejecutando:

```bash
pip install Flask==3.0.0 requests==2.31.0
